## 使用方法

* 插入管道 - 选中相应管道后按Insert键
* 删除管道 - 选中相应管道后按Delete键
* 修改宽度 - 单击选中某根管道，然后修改宽度后按回车即可实时反馈
* 是否开始计算 - 当两根输入管道连接到输出管道后，便计算。同时会用箭头显示管道内流向
